# serhafen Admin Center

This is a modern React + TypeScript SaaS template built with Vite, featuring a complete dashboard interface with internationalization support.

## Features

- **Modern Tech Stack**: React 19, TypeScript, Vite, Tailwind CSS
- **Internationalization**: Built-in i18n support with English and Spanish
- **Responsive Design**: Mobile-first approach with collapsible sidebar
- **Component Library**: Pre-built UI components with consistent styling
- **Dashboard Layout**: Complete admin dashboard with header, sidebar, and main content areas

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
